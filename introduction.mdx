## What is Angga?

Angga provides a centralized hub for your project documentation. You create, edit, and collaborate on docs using a familiar Markdown-based editor. It supports version history, search, and custom navigation to keep your team aligned.

<Callout kind="info">
  Angga integrates seamlessly with your existing workflows, whether you manage open-source projects or internal tools.
</Callout>

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.angga.com/register` with your email.
  </Step>
  <Step title="New Workspace" icon="plus">
    Click "New Workspace" and name it for your project.
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Use the editor to write your `introduction.mdx`.
  </Step>
  <Step title="Publish" icon="globe">
    Hit publish to share with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your docs via API
  const response = await fetch('https://api.angga.com/docs/YOUR_WORKSPACE', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  # Fetch docs via API
  import requests
  response = requests.get(
      'https://api.angga.com/docs/YOUR_WORKSPACE',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
