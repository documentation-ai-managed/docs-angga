## Overview

To start documenting, create a new project in Angga.

## Steps

1. Go to your Angga dashboard.
2. Click "New Project" at the top right.
3. Enter your project name and description.
4. Click "Create".

Your new documentation space is ready. To add pages, select the project and use the "Add Page" button.
